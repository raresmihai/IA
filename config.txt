Configuration steps:

-For this module, you need to have installed the following packages:  "python-duckduckgo-master" , "Google-Search-API-master"  , "beautifulsoup4-4.5.1" 
-For:
		!!! open cmd:
			set PATH=%PATH%;(calea catre  interpretorul 2.7 )

		*"python-duckduckgo-master" :
			with command cd go to python-duckduckgo-master directory,run python setup.py install
		*"beautifulsoup4" enter the following command:
			sudo pip install beautifulsoup4
		*"Google-Search-API-master" :
			with command cd go to Google-Search-API-master directory,run python setup.py install
